# Folder Forge

**by Harish**

A desktop app for creating reusable folder structures. Built for motion designers, video editors, and anyone who sets up the same project folders repeatedly.

## Features

- Create and save folder structure templates
- Visual tree editor — add, rename, delete, drag folders
- Project name modal — rename root folder per project
- Duplicate, export, and import templates as JSON
- Auto-opens created folders in Finder
- Keyboard shortcuts: `Cmd+N` new, `Cmd+I` import, `Cmd+Enter` create, `Cmd+D` duplicate

## Install

```bash
npm install
npm start
```

## Build

```bash
npm run build:mac    # macOS .dmg
npm run build:win    # Windows .exe
```

## Tech

Electron, vanilla JS, no frameworks.
